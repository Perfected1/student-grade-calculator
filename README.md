# Student Grade Calculator (Go)

A simple CLI-based student grade calculator built with Go.  
It takes student scores, calculates total, average, grade, and provides performance feedback.

---

## Features

- Accepts number of subjects
- Collects scores for each subject
- Validates input (0–100 range, no invalid subject count)
- Calculates:
  - Total score
  - Average score
  - Grade (A–F)
- Provides performance remarks based on grade
- Clean CLI interaction

---

## Grading System

| Grade | Score Range |
|------|-------------|
| A    | 70 – 100    |
| B    | 60 – 69     |
| C    | 50 – 59     |
| D    | 45 – 49     |
| F    | Below 45    |
